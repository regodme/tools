- Description: ---------------------------------------

------------------------------------------------------

------------------------------------------------------
- Extract archive in any folder and run program. -----

------------------------------------------------------
- Windows Vista|7|8|10|x86-x64 -----------------------

======================================================
- Launcher settings: ---------------------------------
======================================================
------------------------------------------------------
- After first run will be created file Data\Run.ini. -
- Now, you can configure the following settings: -----
------------------------------------------------------
- Note: The file can be opened with Notepad. ---------
------------------------------------------------------
- ManualCapturePath ------------------- [ yes / no ] -
- allow select folder for captured files -------------
------------------------------------------------------
- AdminCheck -------------------------- [ yes / no ] -
- check administrator rights before launching --------
------------------------------------------------------
- SystemLang -------------------------- [ yes / no ] -
- use auto-detect system language --------------------
------------------------------------------------------
- InstCodecs -------------------------- [ yes / no ] -
- allow to installation of codecs BandiMPEG1 ---------
------------------------------------------------------
- BlockNet ---------------------------- [ yes / no ] -
- block a program access to the Internet -------------
------------------------------------------------------
- RunParam -------------------------------------------
- additional parameters of the run program -----------
